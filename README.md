# File-Explorer
A very simple file explorer written in C++ with Qt.
Currently only works in linux.
## Features
- open in directory with commandline arg
- list ui of directories
- open files by double-click
- open terminal in dir via M2-context menu
- rename/delete files/dirs via M2-context menu
- go to directory via a navbar
- go up one dir via button press
- go forward/backwards with history
- multiple tabs
